# 深度研究参考总索引

> 原先位于根目录 SKILL.md 的“大而全说明书（manual）”已经迁移并拆分到本目录。本文件作为总索引（index），用于把完整方法学材料重新组织为可按需加载（progressive loading）的参考体系。

## 迁移说明

- 原根目录 SKILL.md 现在只承担能力入口（capability entrypoint）职责。
- 原来的长篇方法学内容被拆成三个分册，避免主 skill 在首次触发时把大量理论全部压进上下文。
- 若任务只需要立即执行，请回到根目录的 SKILL.md。
- 若任务需要补理论、补取证规则或补输出模板，请按下面的分册加载。

## 设计原则

- 主入口保持短、快、任务导向。
- 长篇理论不再常驻主文档，只在需要时引用。
- 参考材料按任务阶段拆分，减少无关上下文占用。
